# Sales Performance Dashboard using Tableau

## Project Overview

This project presents an interactive Tableau dashboard designed to analyze sales performance across different regions, segments, products, and time periods. The dashboard provides valuable insights into sales, profit, units sold, and overall business performance.

---

## Tools Used

- Tableau Public
- Microsoft Excel
- Data Visualization
- Basic Statistical Analysis

---

## Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Units Sold
- Profit Margin %

---

## Key Insights

- West region generated the highest profit margin.
- Enterprise segment showed lower profitability.
- Certain products contributed significantly to overall sales.
- Sales and profit trends varied across quarters.
- Top-performing products accounted for a major share of total revenue.
