# 📊 Tableau Visualizations
This project Showcases various data visualizations and analytical dashboards created in Tableau using the Sample Superstore dataset. The dataset contains retail business information from the United States, including sales, profit, products, customers, categories, segments, regions, states, and order dates.

The main objective of this project is to analyze sales performance, evaluate profitability, identify business trends, compare regional and product performance, and generate actionable insights through interactive Tableau dashboards and visualizations.
